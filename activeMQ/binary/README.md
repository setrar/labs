# Install Binary 

```
curl -o apache-activemq-5.16.2-bin.zip https://archive.apache.org/dist/activemq/5.16.2/apache-activemq-5.16.2-bin.zip 
```

```
unzip apache-activemq-5.16.2-bin.zip 
```

```
cd apache-activemq-5.16.2
```

```
bin/activemq start
```


```
bin/activemq stop
```
